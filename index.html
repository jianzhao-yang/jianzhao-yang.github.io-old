《穷查理宝典》读书笔记1-5讲

发表于 2019-03-24 | 分类于 读书笔记

穷查理宝典 - 查理·芒格的智慧箴言录 【1~5讲】

​ [美]彼得·考夫曼 (编)

  1. 要是知道我会死在哪个地方就好了，那我将永远不去那个地方。如果你永远不从别人的失误中吸取教训，那么你的人生不过是一直重复别人的悲惨命运

  2. 逆向思维是最好的思维工具之一。逆向思维意味着不间断的自省、推翻、重建

  3. 铁锤人思维：在手里拿着铁锤的人看来,世界就像一颗钉子。如果不掌握多元化的思维方式，那么我们很容易会陷入思维定势里不能自拔

  4. 做一个可靠的人是至关重要的，而且这是每个人通过努力都可以做到的。靠谱是一个人最重要的品质之一

  5. 你必须把经验悬挂在头脑中的一个由许多思维模型组成的框架上。

  6. 思维模型的来源：科学、工程学、心理学、生物学。虽然每一门学科都足够复杂，但是我们仍然能够很快的学习某一门学科中最重要的一些理论，并从中获益。永远不要理所当然的认为‘自己的专业是某一科，所以没有必要了解其他学科的任何内容’，否则你会因为思维局限和僵化而陷入‘所有的愤怒都来自于自己的无能’的境地而不自知

  7. 规模优势： 成本优势、信息优势、社会认同。

  8. 规模劣势：因规模太大而造成的在非核心领域不够专业的情况。容易被新兴小公司的边缘创新重创

  9. 巴甫洛夫联想（坏信使理论）：当人们听到坏消息，他们会讨厌带来消息的人

    • 人会本能的抗拒自己不想听到话，而不管这些话是真是假、是否对自己有益，如果我们不时常训练自己主动抗拒这种思维，我们就会这样

  10. 你们必须弄清楚自己的优势在哪里，必须在自己的能力圈之内竞争。

  11. 冲浪模型。当冲浪者顺利冲上浪尖，并停留在那里，他能够冲很长很长一段时间。但如果他没冲上去，就会被海浪吞没

  12. 成为赢家的方法是工作、工作、工作、再工作，并期待能够看准几次机会。

  13. 我用一张考勤卡就能改善你最终的财务状况；这张卡片上有 20 格，所以你只能有 20 次打卡的机会——这代表你一生中所能拥有的投资次数。当你把卡打完之后，你就再也不能进行投资了。

  14. 在我看来很明显的是，赢家下注时必定是非常有选择的。

  15. 在净值的水平上，整个投资管理业加起来并没有给所有客户创造附加值。这就是它的运转方式。

  16. 严重警惕意识形态造成的偏见

  17. 许多心理学教材虽然不乏闪光之处，但大体上都是垃圾。如果你不能辨别真伪，那么信息永远不能转化为知识和智慧

  18. 人不能完全避免犯愚蠢的错误，但可以少犯

  19. 剥夺性超级反映综合症：投入越大，越不愿意失去，越要投入。要懂得及时止损

  20. 没有人期望你什么都懂。有些人总是很自信地回答他们其实并不了解的问题，我不喜欢跟他们在一起

  21. 优秀的文学作品需要读者略加思索才能理解。如果你动脑筋才懂得某个道理，你就会更好地记住它。

  22. 一切应该尽可能简单，但不能过于简单。简单是事情的结束，却不应该是开始

  23. 让我来解释一下为什么现在的教育如此糟糕。部分原因在于不同学科之间老死不相往来的现状

  24. 许多工作只要能够让你赚钱，就是体面的工作。

  25. 我把用来对付失望的最佳方法称为犹太人的方法：那就是幽默。

  26. 对一个需要新机器但没有购买的公司来说，它们实际上已经在花钱。

  27. ”看不见的脚“、谢皮科效应：缺乏合理管控的规章制度就是纵容群体犯罪的罪魁祸首

  28. 双轨分析：首先，理性地看，哪些因素真正控制了涉及的利益；其次，当大脑处于潜意识状态时，有哪些潜意识因素会使大脑自动以各种方式形成虽然有用但往往失灵的结论？ 留意自己的主观思维和潜意识，它们常常欺骗自己

《非暴力沟通》读书笔记

发表于 2019-03-24 | 分类于 读书笔记

《非暴力沟通》读书笔记

  • 译序

  • 我曾以为，我的一生将致力于对生命的痛苦做出反应。后来，我发现，这过于沉重、过于灰暗。如果我只看到痛苦，我的心难免会被乌云所笼罩，被绝望所吞没。在徘徊中，终于有一天，我发现，我的人生可以对生命的美丽作出反应。当我看到美——自己、他人以及其他生命的美，我的心中充满了柔情，也找回了生活的热情与活力。

  • 非暴力沟通的四个要素：

    • 观察，即我们此刻观察到什么？清楚地表达观察结果，而不进行判断或评估。

    • 感受，表达此刻观察给自己带来的具体的心理感受。

    • 需要，体会是内心的哪些需要（或价值、愿望等）导致产生那样的感受。

    • 请求，为了改善此刻情况，我的请求是什么。

    • 使用非暴力沟通时，我们要时刻提醒自己不忘对这四个要素的觉察。

Git学习笔记

一、基本操作

【创建版本库】

  1. git init 初始化一个git仓库
  2. git add <文件名> 添加文件到git记录(git不会像svn一样循环扫描文件夹内的变动,性能更好)
  3. git commit -m “提交文件改动的说明” 提交改动
    git commit –amend 补充提交,如果在上次提交时有遗漏或提交信息填写错误,可以使用这个覆盖上次提交
    注意：每次改动之后都要使用git add 提醒git记录,否则无法成功使用git commit提交!
